#### 1-2 Node安装与NPM源切换 [2-1 新的声明方式let（上） (19:36)](https://coding.imooc.com/lesson/444.html#mid=36168)

NRM： npm registry manager,  一个 NPM 镜像源的管理工具

安装： Windows : npm install -g nrm ； Mac : sudo npm install -g nrm

nrm ls : 显示当前可用的镜像源（source）

nrm test  source: 显示镜像源的时间，哪个最快

nrm use source：切换使用某个镜像源

nrm current : 显示当前镜像源

增加定制源： nrm add imooc http://192.168.1,100:7777

删除源：nrm del imooc

### 第2章 ECMAScript2015（ES6）之必备知识

#### 2-1 新的声明方式let（上） (19:36)

新的声明方式：let

* 不属于顶层对象 window 
* 不允许重复声明
* 不存在变量提升
* 暂时性死区
* 块级作用域

delete 只能删除对象属性，而不能删除变量；写 var 是变量，不写 var 是挂载在全局 window 上的属性。
